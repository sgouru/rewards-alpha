Overview

jPList is a flexible jQuery plugin for sorting, pagination and filtering of any HTML structure (DIVs, UL/LI, tables, etc).

  • DIVs Layout Demo
  • Table Demo
  • UL LI Demo
  • Dropdown Filter Demo

Features

  • Works with any content (tables, lists, div elements etc...)
  • Ascending and descending sorting
  • Alphanumeric and numeric sorting
  • Sorting by date
  • Auto pagination
  • Any number of filters (input fields)
  • Any number of drop down filters
  • Supports cookies
  • Fully customizable CSS
  • Works in all major browsers
  • SEO friendly
  • API functions

HTML Structure

1. Add the following HTML to the HEAD section of your page:

<script type="text/javascript" src="js/jquery.js"></script>

<script type="text/javascript" src="js/jplist.min.js"></script>

<link href="css/jplist.css" rel="stylesheet" type="text/css"/>

2. Add plugin call to the HEAD section of your page:

					
<script type="text/javascript">	

$("#demo").jplist({

	//main options
	items_box: ".list", //items container
	item_path: ".list-item", //path to the item
	
	//sort
	sort: {title: "p.title",
		   description: "p.desc",
		   like: "p.like",
		   date: "p.date"},
		   
	//sort options for page load
	sort_name: "title", 
	sort_order: "asc", //"desc",
	sort_type: "text",
	
	//filter
	filter_path: ".filter",
	filter: {title: "p.title",
		 description: "p.desc"}			
});
</script>

How to add sorting filter?

Add the following LI to the sort drop down menu: 

<li><span data-sort="my_sort_filter" data-order="asc" data-type="text">Sort Filter Name</span></li>
  • "data-type" can be "text", "number" or "date"
  • "data-order" can be "asc" or "desc"
  • "data-sort" is a filter name

And add your filter to the plugin: 

$("#demo").jplist({
		
	...
	
	//sort
	sort:  {title: "p.title",
		description: "p.desc",
		like: "p.like",
		date: "p.date",
		my_sort_filter: "jquery path to HTML"},
		   
	sort_name: "title", //default sort name
	sort_order: "asc", //"desc",
	sort_type: "text", //default sort type
	
	...
				
});

How to add search filter?

Add the following INPUT to the section: <div class="filter">...</div>: 

<input data-name="my_search_filter_name" type="text" value="" placeholder="Filter by..."/>

And add your filter to the plugin: 

$("#demo").jplist({
		
	...
	
	filter_path: ".filter",
	filter: {title: "p.title",
		 description: "p.desc",
		 my_search_filter_name: "query path to HTML"},
	
	...
				
});

How to change pagination info style?

To get "1 - 3 of 33" style:

<div class="info" data-type="range"></div>

To get "Page 2 of 45" style:

<div class="info" data-type="page"></div>

Settings List:

					
<script type="text/javascript">	

$("#demo").jplist({

	//main options
	items_box: ".demo-tbl", //items container
	item_path: ".tbl-item", //path to the item
	css_prefix: "jplist", //css class of container	
	redraw_callback: "", //callback function
	no_results: ".jplist-no-results", //path to the "no results" html
	
	//cookies
	cookies: true, //save page state after refresh
	expiration: -1, //cookies expiration in days (-1 = cookies expire when browser is closed)
	
	//sort
	sort: {title: "p.title",
		   description: "p.desc",
		   like: "p.like",
		   date: "p.date"},
	sort_name: "title", //sort name on page load (or "default" for initial page order)
	sort_order: "asc", //default sort order: "asc" or "desc",
	sort_type: "text", //default sort type: "text", "number" or "date"
	
	//filter
	filter_path: ".filter", //path to the filter container
	filter: {title: "p.title",
		 description: "p.desc"},
	
	//paging
	pagingbox: ".buttons", //path to the paging buttons
	pageinfo: ".info", //path to the paging info box
	items_on_page: 3, //items number on page
	paging_length: 7,  //pager length
	show_one_page: false, //if show pager for one button only
	
	//arrows
	prev_arrow: "&lt;",
	next_arrow: "&gt;",
	first_arrow: "&lt;&lt;",
	last_arrow: "&gt;&gt;",
	
	//drop down
	sort_dd_path: ".sort-drop-down", //sorting drop sown menu path
	paging_dd_path: ".page-by",	 //paging drop sown menu path	
	filter_dd_path: ".filter-drop-down"	//drop down filter path
});
</script>

API Functions:

<script type="text/javascript">    
	
	//plugin call
	$("#demo").jplist({				
		....
	});
	
	//API examples ------------------------------------------------------------
	
	//sort by description desc
	$("#sort-desc-test").click(function(){
		var sort_name = "description";
		var order =  "desc"; //"asc" or "desc",
		var sort_type = "text"; //"text", "number" or "date"					
		$("#demo").data("jplist").api_sort(sort_name, order, sort_type);
	});
	
	//sort by likes asc
	$("#sort-likes-test").click(function(){
		var sort_name = "like";
		var order =  "asc"; //"asc" or "desc",
		var sort_type = "number"; //"text", "number" or "date"					
		$("#demo").data("jplist").api_sort(sort_name, order, sort_type);
	});
	
	//change items per page to 10
	$("#page-test").click(function(){
		var items_on_page = 10;		
		$("#demo").data("jplist").api_pages_per_page(items_on_page);
	});
	
	//filter by title with 'car'
	$("#input-filter-test").click(function(){
		var filter_name = "title";
		var value = "car";		
		$("#demo").data("jplist").api_search(filter_name, value);
	});
	
	//jump to the page 3
	$("#page-jump-test").click(function(){
		var page = 3;
		$("#demo").data("jplist").jump_to_page(page);
	});
	
	//filter by category 'food'
	$("#dd-filter-test").click(function(){
		var filter_name = 'food'
		$("#demo").data("jplist").drop_down_filter(filter_name);
	});	
</script>

Minimizing Source Code

For the source code minimization and compilation any tool can be used.
I recommend Google Closure Compiler.
It can be used in the following way:

  1. Install Java on your computer.
  2. Download Google Closure Compiler.
  3. Use the following options:
    java -jar compiler.jar --js source\*.js --js source\blocks\*.js --externs jquery-1.7.externs.js --compilation_level SIMPLE_OPTIMIZATIONS --warning_level VERBOSE --js_output_file ..\jplist.min.js
